1. Download the .jar file. Save it wherever. Your Desktop, maybe.
  2. Open a command line terminal.
    1. In Windows, hold down the Windows key and press R. Type cmd in the Run window.
    2. On a Mac, click the magnifying glass in the top right of your screen and type Terminal.
    3. In Linux, press Ctrl-Alt-T.
  3. Make sure you have the Java Runtime Environment installed by typing java -version. If you don't have it, you can download it here.
  4. Navigate to the folder where you saved the .jar file. If you don't know how to do this, try saving the downloaded .jar file to whatever default location your terminal opens in.
  5. Type java -jar [filename].jar to launch the app.
